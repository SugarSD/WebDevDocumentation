Base Elements

<html>

This element is used to tell the browser where the HTML document starts and where it ends.
Ex:

<!DOCTYPE html>
<html>
<head>
<title>
A Website
</title>
</head>
<body>
<h1>
Hello, World!
</h1>
</body>
</html>
